### About

Computer Engineering student at UFAL (Maceió, Brazil), focused on embedded systems and digital hardware. I like working close to the metal, from register-level firmware to FPGA routing, and I care about picking the right tool for the job over reinventing things that already work well.

Currently doing undergraduate research (PIBIC) at **OptLab**, integrating an ILP-based router with the **VTR** framework for FPGA routing optimization in C++. Also comfortable working with RFID-based embedded systems, including UHF readers, tag encoding, and mobile integration.
